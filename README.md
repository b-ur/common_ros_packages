# orientalmotor_ros
This is a package to run Orientalmotor AR series with ROS.

# Requirement
* Ubuntu 18.04
* ROS melodic

# Installation
```bash
cd ~/catkin_ws/src
git clone https://github.com/shuto1441/orientalmotor_ros.git
cd ../
catkin build
```

# Usage
```bash
roslaunch orientalmotor_ros motor_reconfigure.launch
```
